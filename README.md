# GPX Manager

GPX Manager is an Android application for organizing your trips and locations. It serves as a personal library where you can store and manage GPX route files, PDF documents, and location bookmarks.

## Features

- **Trip Library** -- Organize GPX files and PDF documents in folders. View route details like distance, waypoints, and track counts at a glance.
- **Location Library** -- Save and categorize places with addresses and coordinates. Organize locations in folders for easy access.
- **File Management** -- Upload, download, rename, move, and copy files between folders.

## Requirements

- Android 14 (API 34) or higher
